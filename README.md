# IndicFriendChat (Android) - Voice version

Includes:
- Bot setup (name/photo/state)
- Friend type (female/male/neutral)
- Optional voice:
  - Mic speech-to-text
  - Bot text-to-speech (toggle)
- Indian language detection
- Calls backend POST /chat

Change backend URL in:
app/src/main/java/com/farchase/indicfriendchat/data/net/Network.kt
